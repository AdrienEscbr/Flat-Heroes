Règles du jeu

Les règles sont simples : survivre aux différents niveaux !
Le jeu accepte plusieurs joueurs. Tant qu'un joueur survie au niveau, le jeu continue.

Pour vous déplacer, vous devez utiliser les touches directionnelles. Vous pouvez également sauter sur les obstacles et sur les murs.

Si tous les joueurs sont éliminés, c'est GAME OVER. Mais rassurez vous, il est possible de rejouer le niveau. Pour gagner, il vous suffit de passer tous les niveaux. Mais attention aux redoutables ennemis !

Bon jeu à vous.
